# word_clouds
**Description:** ingests text from pdf(s), processes it, and makes a word cloud(s)
<br>
<br>

**Requirements:**<br>
	PyPDF2, nltk, wordcloud, collection
	wordcloud depends on numpy, pillow, and matplotlib
